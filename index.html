輪動圖片套件示範 bxSlider

在網頁的製作上，時常會有需要多張圖片同時展示的時候。但是受限於螢幕尺寸與版面空間有限，往往無法一次全部展現出來。這時候，輪動式的展示方式或許就會是一個好方法。

CSS 本身就具備 Animation 相關的功能語法，要做出輪播動態並非不可能，這時候只要再加上 JavaScript 的輔助，就可以做出精緻的輪動式圖片效果。

bxSlider

bxSlider 是一款相當老牌的免費幻燈片式輪播套件，由美國程式設計師 Steven Wanderski 所開發。
